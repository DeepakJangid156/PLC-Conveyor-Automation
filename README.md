# PLC Conveyor Automation Simulation

## Project Overview

This project demonstrates the design of an industrial conveyor sorting system using PLC control logic.  
The system simulates a factory automation process where products move on a conveyor and are inspected and sorted automatically.

The goal of this project is to demonstrate understanding of:

- PLC automation logic
- sensor-actuator control systems
- industrial process automation
- system architecture design

---

## System Architecture

Inputs (Sensors)

S1 → Product Detection Sensor  
S2 → Inspection Sensor  

Controller

PLC Controller

Outputs (Actuators)

M1 → Conveyor Motor  
A1 → Sorting Actuator  

---

## Process Flow

1. Product arrives on conveyor.
2. Detection sensor identifies the product.
3. PLC activates the conveyor motor.
4. Product moves to inspection station.
5. Inspection sensor checks product quality.
6. If product is OK → conveyor continues.
7. If product is defective → actuator removes product.
